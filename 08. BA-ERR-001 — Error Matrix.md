# **BA-ERR-001 — Error Matrix**

## **Module: 01\_Auth\_FSD — Authentication**

---

# **1\. Error Handling Principles**

### **ER-PR-01**

Không hiển thị raw error từ backend cho người dùng.

### **ER-PR-02**

Mọi lỗi phải được map sang User Friendly Message.

### **ER-PR-03**

Lỗi validation hiển thị inline tại field.

### **ER-PR-04**

Lỗi API hiển thị global error.

### **ER-PR-05**

Lỗi session phải kích hoạt Session Expired Flow.

### **ER-PR-06**

Mọi lỗi phải được log vào Analytics/Crash System.
